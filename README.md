# CCF 推荐国际学术会议和期刊目录整理

本仓库整理了中国计算机学会（CCF）2022 年推荐国际学术期刊与会议的分领域评级数据，包含简称、全称、出版社/主办方及网址等信息，便于检索与引用。

## 数据来源

- **官方网站**：[中国计算机学会 (CCF)](https://www.ccf.org.cn/)
- **原始文件**：《中国计算机学会推荐国际学术会议和期刊目录-2022.pdf》

本仓库内容基于上述官方目录整理，仅供学习与参考，权威信息请以 CCF 官网及正式发布的 PDF 为准。
